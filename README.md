# Reference Engine

**Reference Engine** is a lightweight Python library for building and simulating mechanical systems using reference frames. The goal of the project is to provide a simple and extensible framework for modeling rigid bodies, coordinate frames, and their interactions in space.

The library is designed as an educational and experimental tool for studying dynamics, with an emphasis on physical concepts such as reference frames, transformations, and equations of motion.

Current features include:

* Definition of hierarchical reference frames
* Basic rigid body representation
* Simple numerical simulations 
* Visualization support for quick testing and validation

The long-term objective is to expand the engine to support more complex dynamical systems, including multibody systems, constraints, and applications in vehicle dynamics.

This project is currently under development.
